# IISPPR Academy - Premium About Section

A world-class, responsive, and highly interactive "About" section built for the International Institute of SDGs & Public Policy Research (IISPPR) Academy. 

This project is designed with a premium 2026 SaaS aesthetic, featuring cinematic parallax scrolling, interactive 3D physics, glassmorphism, and advanced layered animations.

## 🚀 Features

- **3D Tilt & Spotlight Cards**: Interactive glassmorphism cards that tilt on hover with a cursor-following light gradient.
- **Magnetic UI Elements**: Buttons and tags that realistically "pull" towards your cursor using spring physics.
- **Cinematic Parallax Background**: Slowly moving, rotating geometric shapes and glowing light spheres that react to scroll velocity alongside a tech-grid background.
- **Staggered Slot-Machine Animations**: Custom animated text reveals and interactive hover states for modern data presentation.
- **Scroll-Linked Timeline**: A beautiful vertical journey timeline that visually fills up as the user scrolls down the page.
- **Animated Number Counters**: Highlight statistics that dynamically count up from zero when scrolled into view.
- **Custom Spring Cursor**: A floating tech-style cursor replacement on desktop for immersive interaction.

## 🛠 Tech Stack

- **React 18** (Vite)
- **Tailwind CSS v4** (Utility-first styling)
- **Framer Motion** (Advanced physics, spring animations, & scroll tracking)
- **Lucide React** (Crisp, scalable SVG icons)

## 📦 Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Arpit6691/About_section.git
   cd About_section
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```
   The application will be available at `http://localhost:5173`.

## 🎨 Design Philosophy
The design strictly follows a professional academic aesthetic combined with modern tech-startup motion patterns. The color palette focuses on clean whites, soft emerald greens, teals, and sky blues, completely ditching heavy dark modes for a lighter, more approachable, and highly trusted visual hierarchy.
